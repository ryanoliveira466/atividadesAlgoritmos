

## Enunciado 1 - Contagem e multiplicação:
Escreva um programa que conte de 1 a 10 e, para cada número, exiba o resultado da multiplicação desse número por 5.
```bash
for (let i = 0; i < 11; i++){
	console.log(i);
	console.log("Multiplicação por 5: ", i*5);
}
```
